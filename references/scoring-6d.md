# 6D 实战评分模型（港股打新决策核心）

> 本模型脱胎于「HK IPO Sniper」10 年实战方法论，与真实数据引擎（`scripts/hkipo.py`）绑定使用。
> 满分 100 分，逐维评估后得出**四选一决策**。核心原则：**宁可错过，绝不盲打；宁做现金一手，不做亏本孖展。**

---

## D0. 市场水位（Market Environment）— 前提条件 / 一票否决

不算权重，但可**一票否决**。

- **恒指趋势**：大盘在均线上方（多头）还是下方（空头）？流动性枯竭时任何新股都要扣分。
- **新股情绪**：最近 3 只新股首日表现如何？**若处于连环破发潮 → D0 判负，无论总分多高，一律降级为「观望」。**
- 数据来源：web search（近 3 只新股首日/暗盘表现、恒指走势）+ `python3 scripts/hkipo.py sentiment sponsor`。

---

## D1. 基本面（Fundamentals）— 20%

- **赛道稀缺性**：是「港股第一股」还是「第 N 家同质化公司」？
- **业绩增速**：营收/净利复合增长率；若亏损，现金流能烧多久？
- **估值水位**：发行 PE vs 行业平均 PE。**必须有发行折让**，没有 20% 以上安全垫就是耍流氓。
- 数据来源：招股书原文（`pdf2md.py` 精读）+ `aipo ipo-brief`。

| 分档 | 特征 |
|------|------|
| 8-10 | 赛道稀缺 + 高增长 + 发行折让 >20% |
| 5-7 | 细分龙头，增速尚可，估值合理 |
| 2-4 | 同质化，增速平庸，估值偏贵 |
| 0-1 | 夕阳/强监管行业，或估值离谱 |

---

## D2. 保荐人与稳价（Sponsors & Stabilizers）— 15%

**保荐人妖气值分级：**
- **Tier 1（稳）**：大摩 MS、高盛 GS、中金 CICC、中信。
- **Tier 2（中）**：华泰、招银、德意志。
- **Tier 3（妖）**：不知名小券商 → 易出现「围飞」盘，暴涨暴跌，散户慎入。

**绿鞋机制**：是否有绿鞋（超额配售权）？稳价人历史护盘是否卖力？有绿鞋通常是正面信号。

- 数据来源：`aipo ipo-brief`（保荐人/稳价人）+ `sentiment sponsor`（历史胜率、平均首日）。查该保荐人近一年破发率是否 >60%。

---

## D3. 基石投资者（Cornerstone Investors）— 20%

- **锁仓比例**：基石拿走多少货？**>40% 为安全线，>60% 为抢手。**
- **基石成色：**
  - **钻石级**：国家队、主权基金（GIC、Temasek、ADIA、中投）。
  - **黄金级**：顶级私募（高瓴、红杉、景林）、产业链上下游巨头、腾讯/阿里战投。
  - **青铜级**：不知名个人、某地方产业园（凑数的「亲友团」）。
- 数据来源：`aipo cornerstone`（名单/金额/锁定期/占比）。

---

## D4. 情绪与博弈（Sentiment & Game Theory）— 25%（权重最高）

- **孖展超购倍数：**
  - 冷清（<15 倍）：不回拨，货源归机构，利于控盘（但也可能没人要）。
  - 适中（15-50 倍）：常规回拨。
  - 火爆（>100 倍）：回拨 50%。**警惕**：基本面差的小票超购 100 倍，大概率庄家诱多，散户接盘必死。
- **中签率预估**：一手中签率高低决定「摸一手」还是「上乙组」。
- 数据来源：**券商实时预测**（富途/华泰/辉立的一手中签率，最准）；`odds --oversub <倍数> --price <价格>` 仅作粗略初筛，机制B+低入场费热门股会严重低估。

**回拨机制（修正版）：**
1. 常规主板：<15x 不回拨 → 15-50x 回拨 30% → 50-100x 回拨 40% → >100x 回拨 50%（散户踩踏，非龙头视为利空）。
2. 2025.8 新规：机制 A 保留回拨但上限 35%；机制 B 无回拨，招股时预设比例。**需从招股书核实所选机制。**
3. 特殊豁免：二次上市（-S）/ 同股不同权（-W）/ 超大型 IPO，回拨比例较低。

---

## D5. 套利空间（Arbitrage）— 10%

- **AH/美股溢价**：对标 A 股或美股现价，**折扣 < 30% 则吸引力不足。**
- 溢价率 = (A 股价 - H 股发行价) / H 股发行价 × 100%，注意 CNY↔HKD 汇率，看 A 股近期趋势。
- 数据来源：`ah compare <代码> --price <发行价> --name <名称>`。

| A 股溢价率 | 判断 |
|-----------|------|
| ≥50% | 强安全垫 |
| 20-49% | 有一定安全垫 |
| <20% | 无额外优势 |
| <0%（H 反溢价） | 偏贵信号，扣分 |

---

## D6. 成本与盈亏平衡（Cost Breakdown）— 10%

**核心：不算盈亏平衡点，不给结论。** 费率依据富途牛牛官方（2024/10/24 起）：

- **普通申购（现金 / 富途融资）手续费 = 0**；**银行融资申购 = 100 港元/笔**。
- **中签成交费 ≈ 1.0085% × 中签金额**（证监会+财汇局+联交所交易费+经纪佣金），**仅中签成交时收**。
- 融资：**富途年利率 6.8%**、银行以申购页为准；**按日计息、无论中签与否都收**；默认占款 7 天。
- **盈亏平衡线（Breakeven %）** =（中签成交费 + 申购手续费 + 融资利息）/ 一手市值。
- **Rule**：若 Breakeven % > 8% → 触发成本闸门 **[撤退]**。
- 示例：入场费 5,971 HKD 现金申购，盈亏平衡约 +1.0% → 安全垫厚。

---

## 板块特性（评分时叠加判断）

- **18A 生物科技-B**：看核心产品是否进临床 III 期、是否有 License-out 收入；若仅临床 I 期且现金流紧张 → 该维度趋近 0 分。
- **18C 特专科技**：已商业化看营收增速；未商业化看技术壁垒 + 基石阵容。

## 避雷针（Red Flags，命中即扣重分）

- **募资用途**：>30% 用于「偿还银行贷款 / 营运资金」而非研发扩产 → 扣约 20 分。
- **股权结构**：上市前突击大额分红，或大股东持股极度集中（易被洗盘）。
- **保荐人黑历史**：该保荐人近一年破发率 >60%。
- **Pre-IPO 成本极低 + 解禁期短**：上市即面临抛压。

---

## 退出预演（Exit Strategy，报告末尾必给）

- **[竞价跑]**：妖股或超购过热 → 暗盘/开盘竞价即卖出，切忌恋战。
- **[破发跑]**：以「发行价」为防守线，跌破即走。
- **[格局拿]**：顶级基石 + 适度超购 → 可持有至收盘或更久（评分 ≥80 才考虑）。

暗盘（上市前一天 16:15-18:30）：涨幅 ≥30% 可先出 50%；平盘/微跌持有观望；跌 ≥5% 考虑止损。

---

## 最终决策（四选一，结论必须加粗）

| 结论 | 触发条件 | 申购建议 |
|------|---------|---------|
| **全力出击 (All-in)** | 基本面硬 + 折扣大 + 情绪好 | 乙组 / 大额孖展 |
| **现金摸鱼 (Cash Only)** | 质地尚可但不愿付息 | 现金一手 / 多户现金 |
| **防守性申购 (Speculative)** | 有妖股潜质或套利极小 | 白嫖 / 融资一手 |
| **撤退 (Avoid/Skip)** | 全是雷，破发概率大 | 不参与 |

阈值参考：≥70 积极 / 50-69 谨慎 / <50 放弃。**D0 一票否决优先于总分。**

## 附：从「评分卡」到「散户决策器」（第一性原理修正）

6D 单独用是**个股质量评分卡**，回答「这票好不好」。但散户打新的第一性原理是
**「给定本金，最大化期望收益率的概率组合博弈」**，回答「我这点钱该怎么打」。
两者在爆款上会背离：6D 高分的爆款若中签率极低、要顶头槌几百万才中一手，对小散期望≈0。
因此在 6D 之上叠加多层（由 `scripts/breakeven_predictor.py` + `scripts/decision_engine.py` 落地）：

### 第0层：破发概率引擎 D_break（第一道生死闸门，最重要）
打新收益**非对称**：不破发顶多赚多赚少，一旦破发折了夫人又折兵。故「会不会破发」应前置为第一道闸门。
`breakeven_predictor.py` v3 **七因子**加权输出破发概率区间，按第一性原理四层组织（定价根因→首日货源→承接力→市场β→情绪）：
- 因子权重：定价与折溢价(根因)12%、Pre-IPO倍数+老股套现8%、首日流通市值(哑铃U型)15%、货源分散度12%、承接力量(基石×成色+保荐人破发率+绿鞋)18%、市场环境β(同赛道+大盘+VHSI+破发潮，合并根治共线)22%、超购热度(U型)13%。
- 🔴 ≥55% 红灯直接撤退；🟡 40-55% 黄灯仅白嫖一手（结论封顶「现金摸鱼」）；🟢 <40% 绿灯继续。
- **联动**：反推「首日涨幅期望」中值自动填给 D7，消除手填这一最薄弱环节。
- **暗盘 override**：提供 `dark_pool_pct` 时直接覆盖模型预测的首日期望（暗盘是首日最强领先指标），暗盘破发则按破发跑。
- **哑铃与加宽**：首日流通市值两端危险中间安全；微盘(<2亿)只加宽概率区间不上移中心——微盘首日常被拉高诱多、真正杀伤在上市后3-5天阴跌。
- 因子缺失则权重归一并标注 `missing_factors`，绝不脑补。

### 第一层：闸门制（一票否决，先于评分）
打新风险是**非线性、单项致命**的，线性加权会掩盖致命项。任一命中直接 SKIP，无视 6D 总分：
1. 保荐人劣迹（近一年破发率 >60% / 妖庄惯犯）
2. 无实质基石（清一色亲友团 / 无名机构）
3. 疑似老千结构（突击分红 / 股权高度集中 / 关联交易黑洞）
4. 成本致命（盈亏平衡线 >8%）
（注：连环破发潮/大盘急跌已并入第0层破发引擎的大盘水位因子）

### 第二层：D7 期望值引擎（散户真正关心的「划不划算」）
```
一手期望净收益 = 中签率 × 一手市值 × E(首日涨幅%) − 中签成交费(×中签率) − 申购手续费 − 融资利息
期望收益率 ROC = 一手期望净收益 / 占用资金      （这几天赚百分之几）
年化收益率     = ROC × 365 / 占用天数           （折算成一年，便于横向比）
```
- 中签率**以券商实时预测为准**（富途/华泰/辉立的一手中签率）；`odds` 仅初筛，对机制B+低入场费热门股会严重低估（差距可达100倍），不可直接喂 D7。入场费/一手市值来自 `analyze`/招股书；E(首日涨幅) 由破发引擎(第0层)反推。
- 打新占款仅约一周，**用年化更直观**（现金打新常见年化 100%+）。`eval` 同时输出现金/富途融资(6.8%)/银行融资(+100港元)三种方式对比与最优建议。
- 年化揭示「质量高但对散户无效」的爆款背离，这是 6D 看不到的。

### 第三层：组合层（多只认购期重叠时）
- 本金约束下**按年化收益率排序**，期望为负自动剔除。
- **红鞋机制**下小散最优通常是「多只各一手」分散中签，而非重仓单只。
- 用 `hkipo.py calendar` 看资金档期，规划哪几只、各几手、剩余资金留给谁。

> 一句话：**闸门保命 → 6D 判质量 → D7 算划算 → 组合配资金**。这才是对小散户真正有用的打新决策闭环。
